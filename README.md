# PART B — Short Answer Questions (5 × 3 = 15 Marks)

### 1. What is a variable in Python?

**Answer:**

A variable is a named container used to store data in memory. The value stored in a variable can be changed during program execution.

**Example:**

```python
name = "Lucky"
age = 22
```

### 2. What are the rules for naming variables?

**Answer:**

* Variable names must start with a letter or an underscore (_).
* They cannot start with a number.
* They can contain letters, numbers, and underscores.
* Spaces and special characters (such as @, #, %) are not allowed.
* Variable names are case-sensitive.
* Python keywords cannot be used as variable names.

**Examples:**

**Valid:**

```python
name
_age
student1
```

**Invalid:**

```python
1name
my name
@class
```

### 3. What is PVM and what is its role?

**Answer:**

PVM stands for **Python Virtual Machine**. It executes Python bytecode generated by the Python interpreter and converts it into machine-level instructions so that the program can run.

### 4. What is a data type in Python?

**Answer:**

A data type specifies the kind of value a variable stores. It tells Python how the data should be stored and processed.

**Examples:**

* `int` → Integer
* `float` → Decimal number
* `str` → Text
* `bool` → True or False

### 5. Explain primitive and non-primitive data types with examples.

**Answer:**

**Primitive Data Types:**
These store a single value.

Examples:

* `int` → `10`
* `float` → `3.14`
* `bool` → `True`
* `str` → `"Python"`

**Non-Primitive Data Types:**
These store multiple values or collections of data.

Examples:

* `list` → `[10, 20, 30]`
* `tuple` → `(1, 2, 3)`
* `set` → `{1, 2, 3}`
* `dict` → `{"name": "Lucky", "age": 22}`
